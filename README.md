# Chutian Chen - CV Website

Personal website built with [Quarto](https://quarto.org/), featuring my CV and blog posts.

## Local Development

To preview the website locally:

```bash
quarto preview
```

To render the website:

```bash
quarto render
```

## Deployment

The website is automatically deployed to GitHub Pages via GitHub Actions whenever changes are pushed to the main branch.

## Structure

- `index.qmd` - CV homepage
- `blog.qmd` - Blog listing page
- `posts/` - Blog posts directory
- `_quarto.yml` - Configuration file
- `styles.css` - Custom styling
